# Portfolio Redesign - Vision & Objectives

## Objetivo Geral
Criar uma nova versão do portfólio com design minimalista preto e branco, focado em apresentar projetos de forma destacada e demonstrar expertise em AI Engineering através de um AI Agent interativo.

## Design Principles

### Visual Identity
- **Minimalista**: Design clean, muito espaço em branco, tipografia clara
- **Preto e Branco**: Paleta de cores monocromática (preto, branco, tons de cinza)
- **Light & Dark Mode**: Suporte completo para ambos os modos
- **Tipografia**: Foco em hierarquia visual clara e legibilidade

### Estrutura de Conteúdo

#### Hero Section
- Apresentação concisa e impactante
- Nome, título profissional
- Breve descrição destacando 10+ anos de experiência e expertise em AI/ML
- Call-to-action claro

#### Projects Showcase
- **Destaque principal**: Área generosa para mostrar projetos
- Screenshots/imagens dos projetos em evidência
- Descrições claras da tech stack e funcionalidades
- Links diretos para os projetos
- Foco especial no Tasker (AI Project Management Platform)

#### Experience & Skills
- Timeline ou cards para experiências profissionais
- Skills organizadas por categoria (Languages, Frameworks, AI & Data, etc.)
- Layout que facilite scanning rápido

#### AI Agent Widget - DIFERENCIAL
- **Visual**: Chat widget estilo Intercom
- **Localização**: Botão fixo no canto inferior direito
- **Interação**: Ao clicar, abre um chat overlay
- **Propósito**: Demonstrar expertise em AI Engineering
- **Estado atual**: Interface visual apenas (não-funcional por enquanto)
- **Implementação futura**: Integração com Claude API para responder perguntas sobre meu perfil profissional

### Technical Implementation
- Jekyll + Tailwind CSS (mantém stack atual)
- Componentes reutilizáveis
- Responsivo (mobile-first)
- Performance otimizada
- Animações sutis e elegantes

## Inspirações de Design
- Minimalismo suíço (Swiss Design)
- Bauhaus (funcionalidade sobre ornamentação)
- Brutalist web design (simplicidade, tipografia forte)
- Modern portfolios com foco em projetos

## Diferencial Competitivo
O AI Agent chat não é apenas um "contact form" - é uma demonstração prática de habilidades em AI Engineering, mostrando para potenciais empregadores/clientes que sei construir aplicações com IA de forma profissional.
